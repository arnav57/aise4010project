# Instructions to Run
The main file is the .ipynb file, and this contains the entire pipeline for our project,
the dataset used is extracted within this repository for convinience. The Kaggle link is given in the .ipynb file

In order to Run this notebook:

1. Open the .ipynb file in Colab
2. Upload the 'traffic.csv' file into the runtime session
3. In the menubar choose Runtime > Run All
